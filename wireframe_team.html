02 CORE VALUES
Core Values
Bell이 일하는 방식
01
Empathy
우리가 만드는 것의 반대편엔 항상 환자가 있습니다.
Bell의 제품은 데이터나 기능이 아니라, 환자가 일상을 되찾는 경험을 설계합니다. 그래서 우리는 '사용자'가 아니라 '환자'를 먼저 생각합니다.
· 환자의 맥락과 일상을 먼저 이해하기
· 기능보다 경험을 먼저 설계하기
· 다른 전문성을 가진 동료의 시각을 존중하기
02
Integrity
건강과 관련된 일에서 근거 없는 자신감은 위험합니다.
우리가 만드는 제품은 환자의 신체와 생활에 직접 영향을 줍니다. 그렇기에 데이터 앞에서 겸손하고, 틀렸을 때 빠르게 인정하는 것이 Bell의 기본입니다.
· 근거와 데이터를 기반으로 판단하기
· 실수를 빠르게 공유하고 함께 수정하기
· 말한 대로 일하고, 결과로 증명하기
03
Excellence
치료 효과에 "대충"은 없습니다.
임상적으로 검증되지 않은 제품은 환자에게 선택지가 될 수 없습니다. Bell은 높은 기준을 유지하되, 더 나은 방법을 찾는 것을 멈추지 않습니다.
· 안전성과 유효성을 모든 결정의 기준으로 삼기
· 현재 방식에 만족하지 않고 더 나은 방법 찾기
· 과정이 아닌 결과로 판단하기
04
Curiosity
"원래 그렇게 하는 거야"는 Bell에서 통하지 않습니다.
이명 치료가 왜 어려운지, 왜 기존 방식은 한계가 있는지. Bell은 당연해 보이는 것에 질문을 던지는 것에서 시작했습니다.
· 문제의 표면이 아닌 원인을 탐구하기
· 다른 분야의 지식을 연결해 새로운 접근 시도하기
· 모르는 것을 솔직하게 드러내고 질문하기
